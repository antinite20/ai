# 🏠 Quick Reference: Desil Klasifikasi & Criterianya

## Ringkas: 5 Level Ekonomi (Desil 1-10)

### 🔴 DESIL 1-2: MISKIN (LOW INCOME)
```
Rumah Kecil: <25m² | Material: Kayu/bamboo | Atap: Seng berkarat/bocor
Dinding: Tidak dicat, retak dalam | Yard: Tanah, tidak ada pagar
Amenities: TIDAK ADA listrik meter, tangki air, antena
Indikator: Kemiskinan ekstrem
```
**Confidence Setting:** Jika lihat mayoritas tanda di atas → TINGGI

---

### 🟠 DESIL 3-4: BAWAH MENENGAH (LOWER-MIDDLE)  
```
Rumah Kecil: 25-50m² | Material: Mix kayu + bata sederhana
Atap: Seng/asbestos biasa, ada pemeliharaan | Dinding: Semi dicat, retak kecil
Yard: Paved/tanah + pagar sederhana
Amenities: Listrik ada, Tangki air mungkin ada
Indikator: Kelas pekerja
```
**Confidence Setting:** Jika lihat mayoritas tanda di atas → TINGGI

---

```

Yard: Pagar proper, paved, sedikit landscape
Amenities: Meter listrik, Tangki air, Antena mungkin
Indikator: Kelas menengah stabil
```
**Confidence Setting:** Jika lihat mayoritas tanda di atas → TINGGI

---

Amenities: AC visible, Pintu modern, Pencahayaan baik

**Confidence Setting:** Jika lihat mayoritas tanda di atas → TINGGI

## 🔍 Checklist: Cara Analisis Cepat

### **Step 1: Ukuran Rumah**
- [ ] <25m² → Desil 1-2 likely
- [ ] 25-50m² → Desil 3-4 likely
- [ ] 50-100m² → Desil 5-6 likely
- [ ] 100-150m² → Desil 7-8 likely
- [ ] >150m² → Desil 9-10 likely

### **Step 2: Material Utama**
- [ ] Kayu/Bamboo → Lower = Desil 1-4
- [ ] Bata/Concrete → Middle = Desil 5-6
- [ ] Premium/Import → Upper = Desil 7-10

### **Step 3: Amenities Check**
- [ ] No AC, no meter → Desil 1-2
- [ ] Meter only → Desil 3-4
- [ ] Meter + water tank → Desil 5-6
- [ ] AC visible → Desil 7-8
- [ ] Multiple AC + solar → Desil 9-10

### **Step 4: Overall Condition**
- [ ] Neglected, worn → Lower Desil
- [ ] Maintained → Middle Desil
- [ ] Excellent/luxury → Higher Desil

## 💯 Confidence Levels Explained

```
HIGH (80-100%):
→ Evidence jelas, match multiple criteria
→ Ukuran, material, amenities semua consistent
→ Kondisi visible dan clear

MEDIUM (40-79%):
→ Evidence mixed/partial
→ Some criteria match, some unclear
→ Image quality moderate atau ambiguous

LOW (0-39%):
→ Evidence tidak clear
→ Banyak conflicting indicators
→ Image terlalu blur atau angles tidak bagus
```

## 📌 Tips Analisis Akurat

1. **LIHAT SEMUA DETAIL:**
   - Ukuran (penting: 40%)
   - Material kualitas (penting: 30%)
   - Amenities (penting: 20%)
   - Kondisi overall (penting: 10%)

2. **JANGAN ASUMSIKAN:**
   - Lokasi cantik ≠ rumah kaya
   - Rumah besar ≠ kaya (bisa hutang)
   - Hanya lihat VISUAL EVIDENCE

3. **AMBIL KEPUTUSAN BERDASARKAN MAYORITAS:**
   - Jika 3-4 dari 5 indicators match Desil X → itu Desil X
   - Jika conflicting → MEDIUM confidence

4. **SET CONFIDENCE BERDASARKAN CLARITY:**
   - Clear image, multiple matches → HIGH
   - Ambiguous, mixed evidence → MEDIUM
   - Blurry atau unclear → LOW
